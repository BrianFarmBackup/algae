# TL;DR
Algae is a collection of common data structures, intended for use with libraries like [Witchcraft](https://hex.pm/packages/witchcraft)

# Quickstart
Add Algae to your list of dependencies in `mix.exs`:

```elixir

def deps do
  [{:algae, "~> 0.10.0"}]
end

```
